# Andes Construcciones · Horarios (DEMO)

Demo navegable del sistema de horarios, con datos ficticios de una empresa de
construcción. No usa ningún backend real: todas las llamadas a la API se
interceptan en `demo-mock.js` y responden con datos en memoria (se reinician
al recargar la página).

Acceso: desde el [login demo](https://chavezleonardo24-beep.github.io/andes-login-demo/).
